Projects

Gestured Controlled Robot Pet

My Capstone project involved creating a robotic pet using Python, Robotic Operating System (ROS), and Deep Learning technologies. I developed a state machine that allowed the robot to interpret user gestures and perform associated actions, such as following or spinning. In addition, our pet robot was equipped with a camera that could capture pictures of the user. By combining traditional pet actions with modern technology, we created a one-of-a-kind experience that was both entertaining and innovative.

Cache and Paste

My friend and I created a clipboard app that was designed to store up to 5 of the most recent copies in your Windows clipboard. With our app, users could paste any of those 5 items at any time, making it much more efficient than the standard clipboard, which only allows you to store one item. By using Java for the backend and JavaFX for the front end, we were able to create a seamless user experience that was easy to use and navigate. Our app helped users to avoid losing copied items and saved them valuable time when working with multiple copied items at once.

Database and REST API

As part of a university project, I used C# and Entity Framework to create a database to store information related to a hypothetical warehouse, including product details, product locations, orders, and robots in use. To further enhance the user experience, I also designed a REST API that enabled users to create orders, access a list of orders, and view the quantity of products in the warehouse. The API also provided a list of products that were low in stock, ensuring that users were always aware of any potential shortages. By implementing these features, our database and API helped users to better manage and organize their warehouse operations.

Dog vs Cat Binary Classifier

I developed a binary image classifier that uses Python and PyTorch to differentiate between images of dogs and cats. Due to hardware limitations, the model was trained on a small dataset of 1500 images of each animal, resulting in a 70% accuracy rate. The model used seven convolution layers, each activated by a ReLU layer, and two linear layers. To improve the model's performance, I incorporated max pool and dropout layers. Despite the limited dataset, the model was able to accurately classify the images, demonstrating the potential for further improvements with more robust computing power.
